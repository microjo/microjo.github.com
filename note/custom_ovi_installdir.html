自定义诺基亚 Ovi 套件的安装目录

真不知是认为现在用户希望安装什么都不用管最好，还是认为现在硬盘大到不需要关心安装到哪里，诺基亚 Ovi 套件在安装的时候根本没有给用户选择的余地，一律安装到系统盘（C）。以下是从网络上搜索到的自定义安装套件路径的方法：

转载自 How to Change the Installation Directory for Nokia OVI?

if you want to install Nokia Ovi Suite to for example D: drive, to directory "D:\Program files", you could start installation application from command line following way: 举例将诺基亚 Ovi 套件安装到 D:\Program Files 目录下面，在命令行下面输入下面命令进行安装：

Nokia_Ovi_Suite_webinstaller_ALL.exe /TARGET="D:\Program files"

This would create a new folder "Nokia Ovi Suite" under D:\Program files\ and install Nokia Ovi Suite there. 诺基亚 Ovi 套件会在 D:\Program files 目录下面创建新目录 Nokia Ovi Suite，并安装到此目录下面。

经实际使用，确实能达到目的。注意在 Windows Vista 和 Windows 7 下面安装时，必须有管理员权限才能安装。
