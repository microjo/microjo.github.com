自用 User Scripts 分享

用户脚本 (User Script) 能够增强浏览体验，有时能提供意想不到的功能。Chrome 原生支持用户脚本，Firefox 需要安装 Greasemonkey 或 Scriptish 扩展。很多不错的脚本可以在 http://userscripts.org/ 搜索到，以下列出本人自用的：

  1. YouTube HD Suite
    最早接触到的脚本，在 YouTube 页面上显示视频各种格式的下载，用各种颜色标识，直观、方便。

  2. 12306.cn 购票助手
    春节网上购票帮了大忙，自动登陆功能非常有用，还有自动刷新查询票额功能，方便及时发现余票。

  3. get images_BBcode from SkyDrive
    方便提取 SkyDrive 共享图片的外链链接。

  4. DBank Linker
    终于可以不需要注册登录下载华为网盘上的资源了，真是方便 ：）
